# 岳威科技 AI 頂尖實戰課程

一個專業的課程報名網站，展示岳威科技 2026 年 AI 頂尖實戰課程的詳細信息。

## 功能特性

- 響應式設計，支援桌面和移動設備
- 流暢的滾動導航
- 課程時程表展示
- 在線報名表單
- 成功提交反饋彈窗
- 雙語支援（繁體中文 / 日本語）

## 技術棧

- HTML5
- Tailwind CSS
- Lucide Icons
- Vanilla JavaScript

## 快速開始

直接在瀏覽器中打開 `index.html` 即可查看網站。

## 部署

該網站已部署到 Vercel，可通過公開 URL 訪問。
